# 📊 Financial Project Analysis Dashboard

## 📌 Overview
This project presents a complete **financial analysis of a business/project** using Excel and Power BI.  
It includes detailed financial statements and an interactive dashboard to evaluate profitability, cash flow, cost structure, and overall financial health.

The goal of this project is to **move beyond static reports and enable data-driven decision making**.

---

## 📂 Project Structure
📁 Financial-Analysis-Dashboard
│── 📄 Project Report.xlsx (Complete financial model & statements)
│── 📊 Dashboard.pbix (Power BI dashboard file)
│── 📁 Data (Cleaned datasets used for Power BI)
│── 📄 README.md



---

## 🧾 Financial Model Components

The Excel model includes:

- Project at a Glance  
- Cost of Project & Means of Finance  
- Profitability Projections (2026–2030)  
- Cash Flow Statement  
- Balance Sheet  
- Break-even Analysis  
- Depreciation Schedule  
- Financial Ratios  
- Return on Investment (ROI) Analysis  
- Assumptions  

---

## 📊 Power BI Dashboard Features

### 🔹 Key Metrics
- Total Project Cost  
- Loan Amount  
- Return on Investment (ROI)  
- Average Net Profit  
- Closing Cash Balance  

---

### 🔹 Visual Analysis

#### 📈 Profitability Analysis
- Revenue vs Expenses vs Net Profit trend  
- Year-wise net profit growth  

#### 💸 Cash Flow Analysis
- Cash inflow vs outflow  
- Closing balance trend  

#### 🧾 Cost Analysis
- Expense breakdown (Fuel, Salary, Maintenance, etc.)  
- Identification of major cost drivers  

#### 📊 Financial Ratios
- Gross Profit Ratio  
- Net Profit Ratio  
- Current Ratio  
- Debt-Equity Ratio  

---

### 🎯 Key Insights
- Revenue shows consistent year-on-year growth  
- Net profit improves steadily across projection years  
- Power & Fuel is the major cost driver  
- Cash position remains positive throughout the period  
- Low debt-equity ratio indicates lower financial risk
